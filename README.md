# semantic-profile

This project is a simple **semantic-profile** created using **HTML and CSS** as part of Task 4.

The main goal of this task is to understand how multi-page websites work, how navigation is handled using relative paths, and how a common CSS file is reused across multiple pages.

---

# Live Demo
Live Website:
https://ranii1012.github.io/semantic-profile/


##  Pages Included

- Home
- About
- Education
- Skills
- Contact

Each page is created as a separate HTML file and connected using a navigation menu.
